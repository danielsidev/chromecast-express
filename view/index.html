Middleware para Chromecast

Métodos Aceitos

Prefixo: /chromecast
GET => /init

Monta os objetos necessários com os dados da library e inicia as propriedas para o envio do vídeo.

response => json

{success:true, error:false, message:"Modelo Chromecast Iniciado!" }
POST => /sendvideo

param send => json
Informando o nome do Chromecast em device 
        {
         "media": {
              "device":"Sala",
              "playlist":[{
                  "name":"Vídeo de teste",
                  "path":"http://clips.vorwaerts-gmbh.de/big_buck_bunny.mp4"
              }]
            }
          }


Se possuir somente 1 Chromecast não precisa informar o nome. Localizamos o único na rede. 
        {
         "media": {
              "device":"",
              "playlist":[{
                  "name":"Vídeo de teste",
                  "path":"http://clips.vorwaerts-gmbh.de/big_buck_bunny.mp4"
              }]
            }
          }


response => json

{success:true, error:false, message:"Vídeo enviado ao Chromecast: [nome-device]"}
PUT => /playpause

Cada requisição a mesma rota altera o status entre Play e Pause, de acordo com o status atual.

response => json

{success:true, error:false, message:"Play / Pause do Vídeo no Chromecasst" }
PUT => /stop

Para a reprodução do vídeo e mantém a conexão com o chromecast.

response => json

{success:true, error:false, message:"Stop do Vídeo no Chromecast" }
